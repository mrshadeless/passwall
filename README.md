# passwall
Passwall Installation

Run this command in openwrt remote ssh
```
rm -f passwallx.sh && wget https://github.com/mrshadeless/passwall/raw/main/passwallx.sh && chmod 777 passwallx.sh && sh passwallx.sh
```
Done !
